# MC Nga Lyna Profile

Static profile website for MC Nga Lyna, a Vietnamese-English bilingual MC.

## GitHub Pages

Deploy from the `main` branch root. The public entry file is `index.html`.

## Local Preview

```bash
python3 -m http.server 5177 --bind 127.0.0.1
```

Open `http://127.0.0.1:5177/index.html`.
